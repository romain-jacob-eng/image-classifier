# Image Classifier — CNN & CIFAR-10

## Overview
A Convolutional Neural Network (CNN) trained on the CIFAR-10 dataset to classify images into 10 categories (airplane, car, bird, cat, deer, dog, frog, horse, ship, truck) with 72% accuracy. Includes a visual prediction display showing test images alongside their predicted and true labels, color-coded green (correct) or red (incorrect).

## Technologies
Python, PyTorch, torchvision (CIFAR-10)

## How to run
python image_classifier.py
